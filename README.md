# Angular Playground

This project provides a blueprint of an angular application. Different components exemplary illustrate the usage of various concepts/libraries of the Angular ecosystem. 

## Technology Stack
- Angular
- ng-bootstrap / bootstrap
- ngx-translate
- ng-select
